<Readme内の呼称について>

・オブジェクト：ピース、フレームの両方を指します。


<操作方法>

Ctrl + マウススクロール ：拡大・縮小
マウススクロール 　　　 ：縦にスクロール
Shift + マウススクロール：横にスクロール
ESC　　　　　　　　　　 ：オブジェクトの編集を中断

ダブルクリック　　　　　：ダブルクリックしたポイントからオブジェクトの編集開始
クリック（編集中）　　　：オブジェクトの頂点をプロット
Ctrl + Z　　　　　　　　：アンドゥ
Ctrl + Y　　　　　　　　：リドゥ


<その他操作について>

・画面左にアイコンがありますが、一部未実装で動作しません。
・画面左に目の形をしたアイコンがあります。これを押すとプレビュー画面が出ます。
・フレームを編集する場合、プレビュー表示アイコンの下のアイコンをクリックしてください。
　ピースの編集に戻るときはもう一度クリックしてください。
